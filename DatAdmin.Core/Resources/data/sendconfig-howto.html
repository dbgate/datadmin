• If you don't have DatAdmin installed yet (you need version 4.2.5 or newer), download it (for free) from datadmin.com and install
  • Double click on attachment configuration.dca, it will open it in DatAdmin
  • If previous step is not possible with your e-mail client, save attachment to disk and open it from DatAdmin (in menu Options/Import)
  • In tree "Imported configuration" check items, what you want to import
  • Press OK and DatAdmin is ready to use with new configuration
